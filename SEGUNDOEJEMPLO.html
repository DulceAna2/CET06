¿Qué propiedad CSS se utiliza para establecer un contenedor como contenedor flex?

La propiedad CSS que se utiliza para establecer un contenedor como contenedor flex es "flex o inline-flex""

¿Qué valor se utiliza en la propiedad `flex-wrap` para permitir que los elementos se envuelvan en líneas adicionales si no hay suficiente espacio?

El valor que se utiliza en la propiedad "flex-wrap" es "wrap"

¿Cuál es la propiedad CSS que se utiliza para centrar los elementos a lo largo del eje principal horizontal en un contenedor flex?

La propiedad "justify-content:center" se utiliza para centrar los elementos a lo largo del eje principal horizontal

¿Cuál es la propiedad CSS que se utiliza para centrar los elementos a lo largo del eje secundario vertical en un contenedor flex?

La propiedad "align-items:center" se utiliza para centrar los elementos a lo largo del eje secundario vertical

¿Cuál es el propósito del valor `justify-content: space-between` en la propiedad `justify-content` del encabezado?

La propiedad "justify-content" se utiliza para definir el espacio entre y al rededor de los items flex. El valor "space-between" se utiliza para distribuir a los elementos de manera uniforme: al principio, centro y final del contenedor

¿Cuál es el propósito del valor `align-items: baseline` en la propiedad `align-items` del encabezado?

La propiedad "align-items" se encarga de controlar la alineacion de los elementos flexibles en el eje transversal. El valor "baseline" se encarga de alinear los elementos del encabezado a lo largo de la linea de base

¿Qué propiedad CSS se utiliza para establecer el ancho del contenedor del título del header?

Se utiliza la clase "container-title" con la propiedad "width" para establecer el ancho del contenedor del titulos del header

¿Qué propiedad CSS se utiliza para alinear el texto del título en el centro?

La propiedad CSS que se utiliza para alinear el texto del titulo en el centro es "text-align:center"

¿Cuál es la propiedad CSS que se utiliza para establecer un margen a la derecha y a la izquierda del contenedor?

La propiedad CSS que se utiliza para establecer un margen para los cuatros lados es "margin-left" y "margin-right"

¿Qué propiedad CSS se utiliza para establecer el color de fondo del contenedor principal?

La propiedad "background-color" se utiliza para establecer el color de fondo del contenedor
